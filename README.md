# GitHub Profile Finder

A React-based GitHub Profile Finder that allows users to search for GitHub profiles by username and view detailed information about the account. The application retrieves user data from the GitHub API and displays key profile information, including the user's avatar, name, username, bio, location, followers, following count, public repositories, and account creation date. The interface features a modern GitHub-inspired dark theme and provides feedback through loading and error states.

## What I Learned

Through this project, I gained additional experience working with React Hooks, state management, API integration, and conditional rendering. I further strengthened my understanding of handling asynchronous operations with the Fetch API and managing different UI states based on API responses.

In addition, I learned how to use the dependency array in useEffect to trigger side effects when specific state values change, allowing data to be fetched automatically whenever the searched username is updated. I also explored CSS animations using @keyframes to create smooth fade-in effects that improve the user experience.

## Prerequisites

* Node.js (version 18 or later recommended)
* npm (included with Node.js)

## How to Run It Locally

To run this project on your local machine:

* Clone the repository:

  git clone https://github.com/HassanNass/github-profile-finder.git

* Navigate to the project directory:

  cd github-profile-finder

* Install dependencies:

  npm install

* Start the development server:

  npm run dev

Open your browser and visit the URL displayed in the terminal (typically http://localhost:5173 for Vite projects).
